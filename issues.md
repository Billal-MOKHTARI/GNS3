## Incorrect ubridge permissions
```bash
chmod 777 /usr/bin/ubridge

```

## Failed to connect to `localhost:3080`.
- Go to the file `~/.config/GNS3/2.2/gns3_server.conf`.
- Modify the port from `3080` to `3081`.
